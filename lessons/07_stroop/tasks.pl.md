# Zadania — Stroop Challenge

Wykonaj poniższe kroki podczas lekcji. Wszystkie obserwacje i obliczenia zapisuj w zeszycie lub wspólnym dokumencie zgodnie z wskazówkami prowadzącego.

## Krok 1: Uruchom aplikację

Otwórz terminal w katalogu projektu i uruchom:

```
uv run cognitive-data-arcade
```

## Krok 2: Przejdź do Lekcji 07

Użyj klawiszy strzałek, aby przejść do **Lekcji 07 — Stroop Challenge** i naciśnij **ENTER**. Pojawi się krótki ekran wprowadzający. Naciśnij **SPACJĘ** lub **ENTER**, aby przejść do menu lekcji.

## Krok 3: Przeczytaj slajdy teoretyczne w aplikacji

Przed grą przejdź do sekcji **Teoria** w menu lekcji. Przeczytaj wszystkie pięć slajdów — omawiają one automatyczność, trzy warunki, formułę Stroopa i zastosowania kliniczne. Slajdy przedstawiają kluczowe pojęcia w skondensowanej formie.

## Krok 4: Zagraj w jedną pełną sesję Stroop Challenge

Wybierz **Graj** z menu lekcji. Zostaną ci zaprezentowane słowa oznaczające kolory pisane kolorowym tuszem. Twoim zadaniem jest odpowiedź na **kolor tuszu**, nie na znaczenie słowa.

- Używaj klawiszy klawiatury widocznych na ekranie, aby wybrać kolor tuszu.
- Gra przeplata próby zgodne, neutralne i niezgodne.
- Ukończ całą sesję bez przerywania. Staraj się o dokładność, nie tylko o szybkość.

Jeśli szybko skończysz sesję, zagraj po raz drugi — efekty ćwiczenia między sesjami są same w sobie interesujące naukowo.

## Krok 5: Otwórz ekran analizy Stroopa

Wróć do menu lekcji i naciśnij **Z** (lub wybierz **Analiza**, jeśli jest widoczna). Ekran analizy wyświetla średni czas reakcji i dokładność dla każdego warunku (zgodny, neutralny, niezgodny).

Zapisz poniższe wartości w zeszycie:

| Warunek | Średni RT (ms) | Dokładność (%) |
|---|---|---|
| Zgodny | | |
| Neutralny | | |
| Niezgodny | | |

## Krok 6: Oblicz składowe efektu Stroopa

Używając zapisanych wartości, oblicz ręcznie:

```
Interferencja         = RT(niezgodny) - RT(neutralny)    = _____ ms
Facylitacja           = RT(neutralny) - RT(zgodny)        = _____ ms
Całkowity efekt Stroopa = RT(niezgodny) - RT(zgodny)    = _____ ms
```

Porównaj swój całkowity efekt Stroopa z opublikowanym zakresem normatywnym **100–300 ms** dla zdrowych młodych dorosłych.

- Czy Twój efekt mieści się w tym zakresie?
- Jeśli jest większy: czy możesz wskazać czynniki wewnątrzsesyjne (zmęczenie, rozproszenie, szybkość czytania), które mogłyby to wyjaśnić?
- Jeśli jest mniejszy: czy wypracowałeś strategię podczas sesji (np. celowe spowolnienie, aby unikać błędów)?

## Krok 7: Przeanalizuj wzorzec dokładności

Przyjrzyj się wartościom dokładności w poszczególnych warunkach. Odpowiedz na pytania:

1. Czy Twoja dokładność spadła w warunku niezgodnym w porównaniu ze zgodnym? O ile punktów procentowych?
2. Jeśli popełniłeś błędy w warunku niezgodnym, czy były to błędy szybkie (odpowiedź szybsza niż Twój średni RT) czy wolne (odpowiedź wolniejsza niż Twój średni RT)? Błąd szybki sugeruje, że automatyczna odpowiedź z drogi czytania „wygrała" rywalizację; błąd wolny sugeruje, że wykryłeś konflikt, ale mimo to nie zdołałeś go zahamować.

## Krok 8: Porównaj facylitację z interferencją

Masz teraz dwie odrębne wartości: interferencję i facylitację. Która jest większa w Twoich danych?

Badania konsekwentnie wykazują, że interferencja > facylitacja u większości uczestników. Jeśli Twój wzorzec jest odwrócony (facylitacja > interferencja), zastanów się: czy Twój punkt odniesienia neutralny jest właściwy? Czy bodźce neutralne (XXXXX lub słowa niezwiązane z kolorem) okazały się zaskakująco wolne lub szybkie?

## Pytania dyskusyjne

Omów poniższe pytania z grupą lub złóż pisemne odpowiedzi zgodnie z instrukcją prowadzącego:

1. **Dlaczego nie możesz po prostu zdecydować się ignorować słowo?** Użyj pojęcia automatyczności w odpowiedzi.
2. **Jeśli ćwiczyłbyś to zadanie codziennie przez miesiąc, jak zmieniłby się Twój efekt Stroopa?** Co by się zmieniło — ogólny RT, interferencja, facylitacja, czy wszystkie trzy? Dlaczego?
3. **Pacjent z uszkodzeniem płata czołowego jest testowany zadaniem Stroopa. Przewidź wyniki i wyjaśnij je za pomocą modelu dwudrogowego.**
