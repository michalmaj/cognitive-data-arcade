# Lekcja 02: Laboratorium Czasów Reakcji

**Szacowany czas:** 90 minut
**Gra:** ReactionTimeGame — preset Standardowy (60 prób), uruchamiany z menu głównego, należy nacisnąć ENTER na lekcji 02
**Wymagania wstępne:** Lekcja 01 (Big Data w Kognitywistyce)

## Cele kształcenia

- Wyjaśnić, czym jest czas reakcji i dlaczego stanowi okno na procesy poznawcze.
- Opisać metodę odejmowania Dondersa i jej historyczne znaczenie.
- Interpretować histogram czasów reakcji: identyfikować średnią, skośność i wartości odstające.
