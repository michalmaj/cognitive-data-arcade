# Lekcja 03 — Logi zdarzeń i formaty danych

## Cele kształcenia

Po zakończeniu tej lekcji student potrafi:

1. Opisać, czym jest log zdarzeń, i wyjaśnić, jaką informację przechowuje każda standardowa kolumna.
2. Porównać kompromisy między popularnymi formatami danych (CSV, JSON, HDF5, EDF/BDF) i dobrać odpowiedni format do konkretnego zastosowania.
3. Wyjaśnić problem synchronizacji czasu i pokazać, dlaczego niezesynchronizowane zegary mogą unieważnić pomiary czasu reakcji.
4. Wymienić standard BIDS i wyjaśnić, w jaki sposób ustandaryzowane formaty wspierają odtwarzalność badań.

## Aktywność interaktywna

Dla tej lekcji nie ma jeszcze aktywności interaktywnej. Gra eksplorująca formaty danych jest planowana na kolejny sprint.

## Szacowany czas

| Sekcja | Czas |
