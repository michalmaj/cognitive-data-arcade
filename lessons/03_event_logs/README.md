# Lesson 03 — Event Logs and Data Formats

## Learning Goals

By the end of this lesson, students will be able to:

1. Describe what an event log is and explain what information each standard column captures.
2. Compare the trade-offs between common data formats (CSV, JSON, HDF5, EDF/BDF) and select an appropriate format for a given use case.
3. Explain the time-alignment problem and why unsynchronised clocks can invalidate reaction-time measurements.
4. Identify the BIDS standard and explain how standardised formats support reproducibility.
