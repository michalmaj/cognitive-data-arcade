# Lekcja 05 — Brakujące wartości i wartości odstające

## Cele kształcenia

Po zakończeniu tej lekcji student potrafi:

1. Rozróżnić trzy typy brakujących danych (MCAR, MAR, MNAR) i wyjaśnić praktyczne konsekwencje każdego z nich dla analizy danych.
2. Zastosować odpowiednie strategie postępowania z brakującymi danymi, w tym usuwanie wierszy, imputację i wielokrotną imputację.
3. Zdefiniować wartości odstające przy użyciu kryteriów statystycznych (wynik z) oraz wiedzy dziedzinowej (fizjologiczna wiarygodność czasów reakcji).
4. Wyjaśnić, dlaczego średnia jest wrażliwa na wartości odstające w prawostronnie skośnych rozkładach RT, i wskazać alternatywy: medianę i uciętą średnią.
5. Opisać standardy raportowania (APA/CONSORT) dotyczące dokumentowania wykluczonych danych.

## Wymagana aktywność

> **Uwaga:** Do Lekcji 05 nie jest na razie przypisana żadna gra. Mini-gra do czyszczenia danych jest planowana w przyszłej wersji aplikacji. W międzyczasie wykonaj zadania, korzystając z plików CSV generowanych przez Stroop lub Flankera.
