# Lesson 05 — Missing Values and Outliers

## Learning Goals

By the end of this lesson, students will be able to:

1. Distinguish the three types of missingness (MCAR, MAR, MNAR) and explain the practical consequences of each for data analysis.
2. Apply appropriate strategies for handling missing data, including listwise deletion, imputation, and multiple imputation.
3. Define outliers using both statistical criteria (z-score) and domain knowledge (physiological plausibility of reaction times).
4. Explain why the mean is sensitive to outliers in right-skewed RT distributions, and identify alternatives such as the median and trimmed mean.
5. Describe the reporting standards (APA/CONSORT) for documenting excluded data.
