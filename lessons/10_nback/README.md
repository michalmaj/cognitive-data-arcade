# Lesson 10 — Working Memory: N-Back Memory Grid

## Learning Goals

By the end of this lesson, students will be able to:

1. Define working memory and describe the four components of Baddeley's model (phonological loop, visuospatial sketchpad, central executive, episodic buffer).
2. Explain the n-back paradigm and articulate why increasing n increases cognitive load in a non-linear way.
3. Calculate d' from hit rate and false alarm rate for n-back performance and interpret what it measures.
4. Describe the Jaeggi et al. (2008) dual n-back training claim and explain why the scientific community has largely not replicated the far-transfer finding.
5. Identify the neural correlates of working memory load in n-back tasks and explain why bilateral dlPFC activity increases with n.

## Required Activity

This lesson uses the **N-Back Memory Grid** game, available in the application as **Lesson 10**. Complete sessions at 1-Back, 2-Back, and Adaptive mode before answering the analysis questions. At least two Adaptive sessions are recommended to observe the level-adjustment behaviour.

## Estimated Time

|---|---|

## Prerequisites

Lesson 09 (Go/No-Go Guard) is recommended because d' and Signal Detection Theory are introduced there in detail. Students who have not done Lesson 09 should read the d' section in theory.md carefully before proceeding to the analysis tasks.
