# Lesson 06 — Exploratory Data Analysis

## Learning Goals

By the end of this lesson, students will be able to:

1. Explain what EDA is and why it is performed before inferential statistics.
2. Compute and interpret descriptive statistics (mean, median, SD, IQR, skewness, kurtosis) for RT data.
3. Identify the appropriate visualization for a given analysis goal (histogram, box plot, violin plot, Q-Q plot, scatter plot).
4. Explain Anscombe's Quartet and its implications for reporting correlation coefficients without scatter plots.
5. Compare RT distributions across experimental conditions and describe differences in terms of central tendency, spread, and shape.
