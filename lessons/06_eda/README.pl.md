# Lekcja 06 — Eksploracyjna analiza danych

## Cele kształcenia

Po zakończeniu tej lekcji student potrafi:

1. Wyjaśnić, czym jest EDA i dlaczego przeprowadza się ją przed statystyką wnioskową.
2. Obliczyć i zinterpretować statystyki opisowe (średnia, mediana, odchylenie standardowe, IQR, skośność, kurtoza) dla danych RT.
3. Wskazać odpowiednią wizualizację dla danego celu analizy (histogram, wykres pudełkowy, wykres skrzypcowy, wykres Q-Q, wykres rozrzutu).
4. Wyjaśnić Kwartet Anscombe'a i jego implikacje dla raportowania współczynników korelacji bez wykresów rozrzutu.
5. Porównać rozkłady RT między warunkami eksperymentalnymi i opisać różnice pod względem tendencji centralnej, rozrzutu i kształtu.
