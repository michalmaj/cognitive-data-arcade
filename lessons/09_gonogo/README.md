# Lesson 09 — Response Inhibition: Go/No-Go Guard

## Learning Goals

By the end of this lesson, students will be able to:

1. Explain the structure of the Go/No-Go paradigm and describe why the No-Go condition is cognitively demanding.
2. Distinguish between commission errors (false alarms) and omission errors (misses), and explain what each type reveals about attention and impulse control.
3. Apply Signal Detection Theory to compute d' from hit rate and false alarm rate, and interpret what d' measures independently of response strategy.
4. Describe how Go/No-Go performance changes with age and in clinical populations such as ADHD.
5. Compare the Go/No-Go task with the Stop-Signal Task and articulate the conceptual difference between the two inhibitory paradigms.

## Required Activity

This lesson uses the **Go/No-Go Guard** game, available in the application as **Lesson 09**. Complete at least one session at Medium difficulty (75% Go) and one at Hard difficulty (60% Go) before answering the analysis questions.

## Estimated Time

## Prerequisites

Lesson 02 (Reaction Time Lab) is recommended but not required. Students should understand the concept of reaction time and basic error rates. No calculus or statistics background is needed — the z-score table required for d' is provided in theory.md.
