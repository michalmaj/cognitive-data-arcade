# Lesson 08 — Flanker Arena

## Learning Goals

By the end of this lesson, students will be able to:

1. Explain why flanker interference occurs using the attentional spotlight model and the concept of spatial response competition.
2. Calculate the flanker compatibility effect from raw reaction-time data and interpret its size relative to published norms.
3. Describe how the Gratton effect (sequential compatibility effect) reveals that cognitive control is adjusted trial by trial.
4. Identify the Attention Network Test (ANT) and explain how the flanker paradigm is used to dissociate alerting, orienting, and executive control.
5. Predict how time pressure (Easy vs. Hard difficulty) affects the speed-accuracy trade-off and the relative size of the compatibility effect.

## Required Activity

This lesson uses the **Flanker Arena** game, available in the application as **Lesson 08**. Complete at least one Standard session (48 trials) before attempting the analysis tasks. Record your RT and accuracy for both congruent and incongruent conditions from the results screen.
