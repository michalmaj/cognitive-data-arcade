# Zadania — Flanker Arena

Wykonaj poniższe kroki podczas lekcji. Wszystkie obserwacje i obliczenia zapisuj w zeszycie lub wspólnym dokumencie zgodnie z wskazówkami prowadzącego.

## Krok 1: Uruchom aplikację

Otwórz terminal w katalogu projektu i uruchom:

```
uv run cognitive-data-arcade
```

## Krok 2: Przejdź do Lekcji 08

Użyj klawiszy strzałek, aby przejść do **Lekcji 08 — Flanker Arena** i naciśnij **ENTER**. Pojawi się krótki ekran wprowadzający. Naciśnij **SPACJĘ** lub **ENTER**, aby przejść do menu lekcji.

## Krok 3: Przeczytaj slajdy teoretyczne w aplikacji

Przed grą przejdź do sekcji **Teoria** w menu lekcji. Przeczytaj wszystkie pięć slajdów. Opisują one strukturę zadania flankera, formułę efektu flankera, etymologię słowa „flanker", uwagę selektywną i model reflektora Posnera.

## Krok 4: Zagraj w sesję Standard (poziom Easy)

Wybierz **Graj** z menu lekcji. Gdy zostaniesz poproszony o wybranie poziomu trudności, wybierz **Easy** (okno odpowiedzi 3 sekundy).

- W centrum ekranu pojawi się strzałka otoczona strzałkami po bokach.
- Odpowiedz na kierunek **centralnej strzałki** — lewo lub prawo.
- Ignoruj strzałki flankujące, nawet gdy wskazują inny kierunek.
- Ukończ wszystkie 48 prób bez przerywania.

Po zakończeniu sesji zanotuj wyniki pokazane na ekranie.

## Krok 5: Zapisz wyniki — sesja Easy

Zapisz w zeszycie:

| Warunek | Średni RT (ms) | Dokładność (%) |
|---|---|---|
| Kongruentny (flankers zgodne) | | |
| Inkongruentny (flankers niezgodne) | | |

## Krok 6: Oblicz efekt zgodności flankera

```
Efekt zgodności flankera = RT(niezgodny) - RT(zgodny) = _____ ms
```

Porównaj swój wynik z typowym zakresem normatywnym **20–80 ms** dla zdrowych młodych dorosłych.

- Czy Twój efekt mieści się w tym zakresie?
- Jeśli jest większy niż 80 ms: co mogłoby to wyjaśniać? (Rozważ: zmęczenie, nieznajomość zadania, przerwy w uwadze.)
- Jeśli jest mniejszy niż 20 ms: czy wypracowałeś konkretną strategię ignorowania flankers?

## Krok 7: Zagraj w drugą sesję — poziom Hard

Wróć do menu lekcji i wybierz **Graj** ponownie, tym razem wybierając **Hard** (okno odpowiedzi 1 sekunda).

Ukończ pełną sesję 48 prób. Po jej zakończeniu zapisz wyniki w tym samym formacie:

| Warunek | Średni RT (ms) | Dokładność (%) |
|---|---|---|
| Kongruentny (flankers zgodne) | | |
| Inkongruentny (flankers niezgodne) | | |

## Krok 8: Porównaj Easy i Hard

Oblicz efekt zgodności flankera dla sesji Hard i porównaj z Easy:

| Sesja | Efekt zgodności RT (ms) | Różnica dokładności (Kongruentny - Inkongruentny, %) |
|---|---|---|
| Easy | | |
| Hard | | |

Odpowiedz na pytania:

1. Czy Twój efekt zgodności RT zmniejszył się w sesji Hard? Jeśli tak, czy oznacza to, że Twoje wyniki się poprawiły, czy że zachodzi inny proces?
2. Czy różnica dokładności (kongruentny vs. inkongruentny) wzrosła w sesji Hard? To sygnatura kompromisu szybkość–dokładność: gdy jesteś zmuszony do szybszego reagowania, różnice w dokładności między warunkami stają się bardziej wyraźne.
3. W warunku Hard, czy zdarzyło Ci się odpowiadać zanim byłeś pewien, w którym kierunku wskazuje centralna strzałka? Jeśli tak, co kierowało Twoją odpowiedzią?

## Krok 9: Zastanów się nad efektami sekwencyjnymi

Przeanalizuj wzorzec błędów w sesji Hard. Zastanów się, czy zauważyłeś różnicę między próbami następującymi po łatwych (kongruentnych) i trudnych (inkongruentnych) próbach. Nie możesz formalnie obliczyć efektu Grattona bez danych dla każdej próby z osobna, ale możesz mieć subiektywne wrażenie. Zapisz swoją obserwację.

## Pytania dyskusyjne

Omów poniższe pytania z grupą lub złóż pisemne odpowiedzi zgodnie z instrukcją prowadzącego:

1. **Efekt zgodności flankera (20–80 ms) jest znacznie mniejszy niż efekt interferencji Stroopa (100–300 ms). Co różnica wielkości mówi Ci o tym, jak automatyczne są dwa źródła konfliktu?**
2. **Pod presją czasową (poziom Hard) efekt zgodności RT może maleć, podczas gdy efekt zgodności dokładności rośnie. Jak interpretujesz to w kategoriach strategii poznawczej?**
3. **Jak wyjaśniłbyś efekt Grattona komuś, kto nigdy o nim nie słyszał? Użyj analogii z życia codziennego.**
